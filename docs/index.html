RIMAPI

概要

RIMAPI は RimWorld に REST API サーバーを追加するモッドで、ゲームのデータを外部から取得・操作できます。コロニーの監視、ポーン操作、資源管理などを自動化できます。

主な特徴

  • RESTful API — 標準的な HTTP エンドポイント
  • リアルタイムイベント — Server-Sent Events (SSE)
  • 拡張可能 — 他のモッドがエンドポイントを追加可能
  • 非ブロッキング / 安全 — メインスレッド上で適切に処理
  • 自動ドキュメント — 自己記述的な API

使い始め（概要）

  1. RIMAPI モッドを RimWorld に導入する。
  2. ゲームを起動して RIMAPI の HTTP エンドポイントにアクセスする。
  3. API ドキュメントを参照して必要なエンドポイントを使用する。
